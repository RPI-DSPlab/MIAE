# Experiments

This directory contains the code for different experiments. 

## `Experiments`

### `experiment/testing-mia`

This folder contains scripts for testing the MIA. It's used to test the performance of the MIA on different datasets and models. They are also serving as examples on how to use the MIAE package.

### `sample-metrics-eval`
This folder contains scripts for evaluating the sample metrics. It's used to compare the performance of different sample metrics on different datasets and models. They are also serving as examples on how to use the MIAE package.

### `detection-rate-vs-example-hardness`
This folder contains scripts for evaluating the detection rate of the MIA on different datasets and models. It's used to compare the performance of the MIA on different datasets and models. They are also serving as examples on how to use the MIAE package.

Please ensure that all necessary dependencies are installed and the environment is properly set up before running the experiments.

## Dependencies

This project is implemented in Python and uses several libraries such as `torch`, `numpy`, and `os`. Please ensure these libraries are installed in your Python environment before running the experiments.
