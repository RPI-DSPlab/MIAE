# Experiments

This directory contains the code for different experiments. 


### `experiment/testing-mia`

This directory contains scripts for testing the MIA. It's used to test the performance of the MIA on different datasets and models. They are also serving as examples on how to use the MIAE package.

### `experiment/mia_comp`

This directory contains scripts for the paper *Membership Inference Attacks as Privacy Tools: Reliability, Disparity and Ensemble*.
